# Trello Clone Practice
[![Maintainability](https://api.codeclimate.com)](https://codeclimate.com)

### Описание проекта
Учебный проект по разработке веб-приложения для управления задачами (клон Trello). Реализована функциональность создания досок, списков и карточек с возможностью их перемещения.

### Стек технологий
- **Frontend:** React.js
- **State Management:** React Hooks / Context API
- **Библиотеки:** react-beautiful-dnd (для Drag-and-Drop)
- **Стилизация:** Styled-components / CSS

### Как запустить локально
1. Установите зависимости: `npm install`
2. Запустите проект: `npm start`
3. Откройте `http://localhost:3000` в браузере.
